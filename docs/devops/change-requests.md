# Dev → DevOps Change Requests

Log for requests filed by Dev (or PO) when an infra / CI / deploy change is
needed. DevOps triages, writes a runbook/spec, and implements. Dev never edits
docs/devops/ or infra/ directly.

Format per entry:

    ### CR-DEVOPS-### — <summary>
    - Requested by: <name> (Dev), <date>
    - Area: <CI | deploy | env var | infra | storage>
    - Status: open | in-progress | done | declined
    - Context: <why this is needed>
    - Ask: <the concrete change>
    - Resolution: <filled when done — link to runbook / pipeline change>
